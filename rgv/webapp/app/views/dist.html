Distance analysis tool

This module gives the possibility to perform a distance analysis between one selected toxicogenomics signatures and all other TOXsIgN toxicogenomics signatures.

As many existing procedures, the TOXsIgN distance analysis tool is based on the euclidean and correlation distance.

How to perform enrichment

Input data: Select the signature using the "Select signature" selector
Parameters: Defined which parameter to use as threshold and the threshold value
Output data: : Table that corresponding to the distance between the selected signature and all other TOXsIgN public signatures (including your own private signatures), where N is the total number of HomoleGene Ids measured, R is the total number of HomoleGene Ids meeting the criterion, n is the total number of HomoleGene Ids in this specific signature, and r is the number of HomoleGene Ids meeting the criterion in this specific signature and ratio corresponding to r/union(n,R).
