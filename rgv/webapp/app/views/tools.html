Search

Search engineuse

The search engine supports complex fielded queries to retrieve relevant data and associated signatures according to various experimental parameters, observed toxicological effects and/or up/down-regulated gene lists. This module is especially relevant for identifying comparable experiments (i.e. performed on similar models and/or with close environmental factors) or experiments with similar outcomes (i.e. that resulted in similar physiological, molecular and/or omics effects).

Browse ontologiesuse

The TOXsIgN ontologies tool allows users to browse the ontologies used to describe data in TOXsIgN. This feature is especially relevant for filling the excel template used to insert data in TOXsIgN. This tool use NCBI OntologyTree widget from BioPortal. This widget is an embeddable Flex application that allows a user to interact with a tree browser for ontologies. The widget is configured to display a list of all the ontologies in TOXsIgN, allowing the user to select which one to display, or can be setup to view a particular ontology.

Convert

Conversion tooluse

The TOXsIgN conversion tool convert a list of EntrezGene Ids or HomoloGene Ids to HomoloGene Ids or EntrezGene Ids. .

Compare

Compare signaturesuse

The TOXsIgN compare module, is a feature allowing users to compare toxicogenomics signatures between them or in several custom lists. This module is adapted from P. Bardou et al.. It works and displays intersection between up to 6 signatures in a Venn diagram.

Distance analysisuse

This module gives the possibility to perform a distance analysis between one selected toxicogenomics signatures and all other TOXsIgN toxicogenomics signatures. As many existing procedures, the TOXsIgN distance analysis tool is based on the euclidean and correlation distance.

Analyse

Functional analysisuse

This module gives the possibility to perform enrichment in biological process, molecular function, cellular component and phenotype for a selected signature. The tool return all over/under represented term associated to the selected signature. To be able to compare these results across species, each EntrezGene Id (which composed the signature) has been converted to HomoloGene Id (deprecated EntrezGene Id(s) are not used).

Extract

Extract toxicogenomics signature (under construction)

This module allow users to extrat the toxicogenomics signature from microarray analysis data. Its return a list of up-regulated genes, down regulated genes and all interogated genes.
This tool is under construction and will be available soon (2017).

Predict

ChemPSy - Prediction tool (under construction)

The prediction tools are based on the ChemSPY project developed within the IRSET institut. The aim of ChemPSy (Chemical Prioritization System) is to provide an innovative tool based on several bioinformatics and biostatistics methodologies to analyze and integrate massive toxicogenomics datasets. Specific objectives include: (1) classification of chemicals based on transcriptional signatures, i.e. the set of genes whose expression is known to be positively or negatively altered after an exposure to these compounds; (2) the association of classes with human pathologies or deleterious phenotypes, i.e. classes containing toxicants with well-known effects; (3) the prediction of novel reprotoxicants and/or endocrine disruptors based on transcriptional signature similarities with known chemicals affecting testis development and function.
